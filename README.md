### Setup

Navigate to directory where the code script is located

```sh
$ cd ~/FileLibrary
```
Install the dependencies and devDependencies.Run Following command after navigating to directory.

```sh
$ rvm use 3.0.0
````

```
$ bundle install
```



### Setup DB.

```
rails db:create
```
```
rake db:migrate
```



### Steps to run Application

Start the rails server
```sh
$ FileLibrary/ rails s
```
